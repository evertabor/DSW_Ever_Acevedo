Display: inline-block

El elemento inline-block se comporta como un elemento de bloque y un elemento de linea, es decir tiene ambas propiedades. Cuando un elemento tiene esta propiedad solo se pondra al lado del otro elemento, si ese otro elemento es de tipo inline o inline-block. Cuando no haya espacio para que ambos elementos puedan estar en linea, es decir que no quepan ambos en una sola linea, estos tomará su comportamiento como un elemento block y se pondran en una sola linea cada uno.
