Display: none

La propiedad display puede tener el valor de none el cual nos permite desactivar la visualizacion de un elemento; cuando se utiliza none, todos los elementos descendentes también quedan desactivados. El documento se procesa como si el elemento no existiera en el árbol de documentos.
