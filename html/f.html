Propiedad display CSS

La propiedad CSS display especifica si un elemento es tratado como es tratado un elemento en bloque (block), un elemento en línea (inline) o un elemento de bloque en linea (inline-block).

La propiedad display puede tener el valor de none el cuel nos permite desactivar la visualizacion de un elemento; cuando no se utiliza none, todos los elementos descendentes también quedan desactivados. El documento se procesa como si el elemento no existiera en el árbol de documentos.

La propiedad display toma los siguientes valores:

  • display:block
  • display:inline
  • display:inline-block
  • none
    •

display:block

Un elemento a nivel de bloque siempre comienza en una nueva línea y ocupa todo el ancho disponible (se extiende hacia la izquierda y hacia la derecha tanto como puede).

El elemento div es un elemento a nivel de bloque.

Ejemplos de elementos a nivel de bloque:

  • div
  • h1
  • p
  • form
  • header
  • footer
  • section

display:inline

Un elemento en línea no comienza en una nueva línea y solo ocupa el ancho necesario.

Ejemplos de elementos en línea:

display:inline-block

EL elemento inline-block se comporta como un elemento de bloque y un elemento de linea, es decir tiene ambas propiedades. Cuando un elemento tiene esta propiedad se pone al lado de un elemento en liena e intenta tomar todo el ancho que se posible.
